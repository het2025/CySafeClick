# 🔒 Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes     |

## Reporting a Vulnerability

If you discover a security vulnerability in CySafeClick, **please do NOT open a public GitHub issue**.

Instead, report it responsibly via:

- 📧 **Email:** security@cysafeclick.app
- 🔐 **Subject line:** `[SECURITY] Brief description of the issue`

### What to include in your report:
- Description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any suggested fix (optional)

### What to expect:
- **Acknowledgement** within 48 hours
- **Status update** within 7 days
- **Credit** in the release notes if you wish (responsible disclosure)

We take all security reports seriously. Thank you for helping keep CySafeClick and its users safe! 🛡️
